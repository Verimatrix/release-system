What's new

Component option for the Router ItemID behaviour. Set the new option “Use ItemID to build SEF URLs” to Yes if you are using this component on your own site. The default behaviour of the ARS Router is not the typical way Joomla component routers work for reasons that have to do with the way we want it to work on our site.

Bug fixes and minor improvements. Please take a look at the CHANGELOG below.

Joomla 4 only

This version will only run on Joomla 4. We will stop maintaining a Joomla 3 version for this software by the end of 2021.
